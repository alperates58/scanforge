# Non-Goals

ScanForge şunları yapmayacak:

- İzinsiz domain taraması
- DDoS/stress test
- Kimlik bilgisi brute force
- Exploit zinciri çalıştırma
- Veri sızdırma denemesi
- Web shell yükleme
- Yetkisiz admin panel denemeleri
- Üçüncü taraf hedeflerde agresif tarama

Amaç güvenli doğrulama, risk tespiti ve düzeltme rehberidir.
