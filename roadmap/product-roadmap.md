# ScanForge Product Roadmap

## Vision
ScanForge, web sitesi sahiplerinin tek panelden güvenlik, domain, DNS, SSL, teknoloji ve bilinen zafiyet durumunu görmesini sağlayan premium bir güvenlik denetim platformudur.

## Core Promise
- URL gir.
- Domain sahipliğini doğrula.
- Keşif yap.
- Teknoloji tespit et.
- Teknolojiye uygun güvenli tarama motorlarını çalıştır.
- Bulguları normalize et.
- DeepSeek ile anlaşılır rapor oluştur.
- Geçmiş taramalarla karşılaştır.
- Kritik açık varsa uyar.

## MVP
- Auth
- Website ekleme
- DNS TXT veya HTML doğrulama
- Passive discovery
- Technology fingerprinting
- SSL/Header/Cookie/DNS scan
- Nuclei safe template scan
- ZAP baseline scan
- AI summary
- Dashboard
- Report export HTML
- Coolify deploy

## V1
- Scheduled scans
- PDF reports
- WordPress özel tarama
- Subdomain inventory
- Finding deduplication
- AI chat over scan findings
- Email/Telegram alerts

## V2
- Authenticated scan
- Cookie/session analyzer
- Compare scans
- Team/multi-tenant
- API keys
- White-label reports

## V3
- Compliance mapping
- OWASP ASVS checklist
- CI/CD scan gates
- GitHub integration
- Advanced false-positive triage
