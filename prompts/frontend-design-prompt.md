# Frontend Design Prompt

Design a premium SaaS dashboard for ScanForge.

Style:
- Clean white interface
- Soft gray borders
- Blue accent
- Large score cards
- Elegant sidebar
- Security product feel similar to Linear/Vercel/Apple, not cyberpunk

Screens:
- Dashboard
- Websites
- Add Website
- Website Detail
- Scan Detail
- Vulnerability Detail
- AI Analyst
- Reports
- Settings

Use React + TypeScript + Tailwind + shadcn/ui.
