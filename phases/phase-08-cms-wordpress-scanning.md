# Phase 08 - CMS and WordPress Scanning

## Goal

Tespit edilen CMS/framework icin hedefli ama guvenli kontroller ekle.

## Deliverables

- WordPress safe detection and version checks.
- Laravel/PHP/ASP.NET/Node targeted safe checks.
- WPScan placeholder or adapter with token config.
- CMS finding normalization.

## Acceptance

- Brute force veya credential test yoktur.
- CMS scans only run when fingerprint confidence is sufficient.
