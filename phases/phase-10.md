# Phase 10 - CMS and Framework Specific Scanners

## Goal
Tespit edilen teknolojiye göre hedefli kontroller.

## Examples
- WordPress -> WPScan, wp-content, plugin/theme enumeration safe mode
- Laravel -> .env exposure, debug, ignition, storage links
- PHP -> phpinfo, exposed composer files, backup files
- ASP.NET -> headers, trace, config exposure
- Nginx/Apache -> misconfiguration templates

## Acceptance
- Technology-to-scanner matrix çalışır.
