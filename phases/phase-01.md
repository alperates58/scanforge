# Phase 01 - Product Foundation

## Goal
Repo, stack, dashboard shell ve temel proje iskeletini kur.

## Tasks
- Laravel/Node backend veya seçilen backend iskeleti
- React + Vite + TypeScript frontend
- PostgreSQL + Redis
- Docker Compose
- Basic auth screens
- Premium layout: sidebar, topbar, cards

## Acceptance
- `docker compose up` ile frontend/backend açılır.
- `/dashboard` boş ama premium shell hazırdır.
- `.env.example` eksiksizdir.
