# Tooling List

- Nuclei: template/CVE/misconfig scan
- OWASP ZAP: baseline scan
- httpx: HTTP probing
- dnsx: DNS records
- subfinder: subdomain discovery
- katana: crawler
- arjun: parameter discovery
- testssl.sh: TLS analysis
- WhatWeb/Wappalyzer: technology detection
- WPScan: WordPress-specific checks
- Trivy: source/container/dependency when source is available
- Lighthouse optional: performance/accessibility later
